Appendix 1

The Business Carbon Calculator by Normative is a carbon calculator, meaning it helps businesses estimate their carbon footprint. It is a service where a user can create a user account and attach it to a business. Through the service the user is then also able to provide data related to both financial and non-financial business activities including but not limited to activities such as electricity and heating. The data is provided for specific time periods and carbon emissions are measured for the corresponding time periods based on the activity data provided. The user can update the data and recalculate emissions, add new time periods, and see their calculated emissions results on a homepage.

This agreement only covers usage of the service “Business Carbon Calculator by Normative” and shall not be confused with any other services offered by Normative. The service is a web application accessible through a web browser such as Mozilla Firefox or Google Chrome. The associated API endpoints used by the web application may only be called through the web application. Any other API endpoints available at the API’s but not called by the web application are not included as part of the described service.
